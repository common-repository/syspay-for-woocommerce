Syspay payment gateway plugin for WooCommerce.

== Description ==

Easy payment solution for your Wordpress website:

 * This plugins provides the syspay payment gateway for WooCommerce
 * Accept online credit card payments on your online shop

== Installation ==

1. Install WooCommerce plugin on your Wordpress site and enable it.
2. Then install the Syspay plugin
3. Create an account on https://app.syspay.com, to get your merchant credentials for enabling the payment gateway.
4. You can configure the payment providers order in the WooCommerce settings.
